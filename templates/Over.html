The HOME page
When you enter the home page you can see the button group Manuel control. Beneath the group you can find two buttons manual and automatic these buttons control how the greenhouse works. 4

When the state is set to automatic the buttons above don’t work because the greenhouse is doing watering, lighting and cooling automatically and therefore you can’t change the state of these things.

When the state is set to manual you can control the greenhouse manual with these buttons above. You can turn on the fan or close the roof as you will.

Beneath the manual buttons you can find the value of every sensor its currently reading. And finely you can find 4 table with data it collected every x seconds. (For testing purpose it will take a reading every 5 seconds. In real, you can set it on 1 hour per reading.

The DETAILS page
On top, you can find one button per sensor. When you choose one of these buttons you can see the details in a graph of the sensor. The sensor you currently watching to is the sensor that is highlighted on top.

The SETTINGS page
When the greenhouse is set to automatic you can change these parameters.

The first one controls the temperature. When the temperature is above the temperature you set it at the greenhouse will automatic open the roof and use the fan to get the temperature to what you set. Also, the led change to red because it’s too hot.

The second one controls the humidity. When the humidity is beneath the number you set it at the greenhouse will automatic give the plants water. It will stop when the humidity is bigger than the set humidity. Also, the led will change to blue because the greenhouse is giving the plants water.

The page CLEARDATABASE and SHUTDOWN
When you press cleardatabase all the saved data will be deleted out of the database.

When you press shutdown, the server will close and the greenhouse will stop working.
